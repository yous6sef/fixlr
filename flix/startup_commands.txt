TROUBLESHOOTING:
================

Database Connection Error:
  - Verify PostgreSQL is running on Neon cloud
  - Check db.php for correct credentials
  - Connection string: ep-long-pond-agq9ju7z-pooler.c-2.eu-central-1.aws.neon.tech

Port Already in Use:
  - Use different port: php -S localhost:8001
  - Kill existing PHP process: taskkill /F /IM php.exe

Database Already Initialized:
  - You can run db_schema.php multiple times (it won't duplicate data)
  - To reset database, manually delete all tables and run again
